## 🛠️ Tecnologie Utilizzate

Per la realizzazione di questo portfolio ho scelto strumenti moderni per garantire velocità e reattività:

* **HTML5 & CSS3:** Struttura semantica e design personalizzato (niente template pronti!).
* **JavaScript (ES6+):** Per le animazioni fluide e la gestione della navigazione.
* **Font Awesome:** Per l'iconografia.
* **Google Fonts:** Per una tipografia leggibile e moderna.

---

## ✨ Caratteristiche Principali

- [x] **Design Responsive:** Ottimizzato per desktop, tablet e smartphone.
